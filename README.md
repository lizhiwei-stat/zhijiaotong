# 职教通 | Zhijiaotong

> 面向职业院校的 AI 教学助手 Agent 系统

## 项目愿景

用开源大模型（Qwen2.5-7B）+ LoRA 微调 + RAG + 多 Agent 协作,
帮职业院校一线教师自动批改编程作业、按大纲出题、生成教案。
数据不出校园, 本地离线部署。
